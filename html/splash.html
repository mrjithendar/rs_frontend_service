Welcome to DKODE's Robot Shop

Here you will find all of DKODE's products. Have a browse around and see whats here.

This is a simple microservices ecommerce application. This application is for learning AWS DevOps. It has been built using various technologies:

  • AngularJS
  • Python
  • Nginx
  • NodeJS
  • Java
  • Golang
  • MongoDB
  • Redis
  • MySQL
We will deploy this application into aws using tools like Python, Ansible, Terraform, Packer, Jenkins, Kubernetes. We will enable autoscaling based on the load testing and monitor metrics by Prometheus, Gragana and more.

To find out more visit the Decode DevOps site.
